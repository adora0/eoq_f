EOQ - Calcolo Lotto Economico di Acquisto


Applicazione per il calcolo del lotto economico di acquisto - EOQ

Premere su EOQ per inserire i parametri di calcolo.
I dati possono essere inseriti manualmente o caricando un file esterno - Il file supportato è di tipo Comma Separated Value con delimitatore ';'

Struttura file .csv anno,costo unitario prodotto, costo setup ordine, costo mantenimento, valore domanda
Per testare l'applicativo è possibile generare dati random.
